🎨 ÉTAPE 1 : GÉNÉRER TON SITE AVEC CLAUDE AI

1.1
Se connecter à Claude AI
  1. Va sur https://claude.ai
  2. Connecte-toi avec ton compte
  3. Clique sur "New Chat" pour commencer une nouvelle conversation
1.2
Le prompt magique pour générer ton site

Copie-colle exactement ce prompt dans Claude AI (remplace les éléments entre [crochets] par tes informations) :

Je veux que tu crées un site web moderne et professionnel pour moi. Voici mes informations : - Nom/Entreprise : [TON NOM OU NOM DE TON BUSINESS] - Activité : [CE QUE TU FAIS - ex: Coach, Consultant, Service, etc.] - Description courte : [1-2 phrases sur ce que tu proposes] - Couleur préférée : [ex: bleu, vert, rouge, ou laisse Claude choisir] - Contact WhatsApp : [TON NUMÉRO] - Email : [TON EMAIL] Crée-moi une landing page complète avec : - Design moderne et attractif - Section hero avec titre accrocheur - Mes services/offres - Témoignages (tu peux en inventer 2-3) - Section contact - Footer - Responsive (mobile et desktop) - Animations CSS subtiles Le code doit être en HTML, CSS et JavaScript dans un seul fichier. Assure-toi que tout fonctionne parfaitement.
1.3
Récupérer et sauvegarder le code
  1. Claude va te générer un code HTML complet
  2. COPIE TOUT LE CODE (du <!DOCTYPE html> jusqu'à </html>)
  3. Ouvre le Bloc-notes (Windows) ou TextEdit (Mac)
  4. Colle le code
  5. Sauvegarde le fichier sous le nom index.html sur ton Bureau
⚠️ IMPORTANT : Le fichier DOIT s'appeler exactement index.html
1.4
Personnaliser le contenu

Si tu veux modifier quelque chose, retourne sur Claude et dis :

Peux-tu modifier [ÉLÉMENT À CHANGER] dans le site ? Par exemple : "Peux-tu changer la couleur principale en rouge ?"

Claude te donnera le code mis à jour.

🌐 ÉTAPE 3 : HÉBERGER SUR GITHUB PAGES

3.1
Créer un repository GitHub
  1. Va sur https://github.com
  2. Connecte-toi à ton compte
  3. Clique sur le bouton vert "New" (en haut à gauche)
  4. Nomme ton repository : [ton-nom].github.io
    Exemple : boris-site.github.io
  5. ✅ Coche "Public"
  6. ✅ Coche "Add a README file"
  7. Clique sur "Create repository"
3.2
Uploader ton fichier
  1. Dans ton nouveau repository, clique sur "Add file" → "Upload files"
  2. Glisse ton fichier index.html dans la zone ou clique "choose your files"
  3. Ajoute un message : "Premier site web"
  4. Clique sur "Commit changes" (bouton vert)
3.3
Activer GitHub Pages
  1. Dans ton repository, clique sur l'onglet "Settings"
  2. Descends jusqu'à la section "Pages" (dans le menu de gauche)
  3. Dans "Source", sélectionne "Deploy from a branch"
  4. Sélectionne "main" branch
  5. Laisse "/ (root)"
  6. Clique sur "Save"
3.4
Accéder à ton site

🎉 Félicitations ! Ton site est maintenant en ligne !

  • URL de ton site : https://[ton-nom].github.io
  • Exemple : https://boris-site.github.io
⏱️ Attention : Ça peut prendre 5-10 minutes avant que le site soit accessible.
3.5
Mettre à jour ton site

Pour modifier ton site plus tard :

  1. Modifie ton fichier index.html avec Claude
  2. Va dans ton repository GitHub
  3. Clique sur index.html
  4. Clique sur l'icône ✏️ (Edit)
  5. Remplace tout le contenu par ton nouveau code
  6. "Commit changes"

🌍 ÉTAPE 4 : OBTENIR UN NOM DE DOMAINE GRATUIT

4.1
S'inscrire sur Freenom
  1. Va sur https://freenom.com
  2. Clique sur "Sign Up"
  3. Créé ton compte (utilise une vraie adresse email)
  4. Confirme ton email
4.2
Chercher un domaine disponible
  1. Dans la barre de recherche, tape le nom que tu veux
    Exemple : monsite, boris-web, mon-business
  2. Clique sur "Check Availability"
  3. Choisis une extension gratuite : .tk, .ml, .ga, .cf
4.3
Enregistrer ton domaine
  1. Clique sur "Get it now!" à côté du domaine choisi
  2. Sélectionne "Use DNS" → "Use custom DNS"
  3. Dans les champs DNS, mets :
    • DNS 1 : 185.199.108.153
    • DNS 2 : 185.199.109.153
    • DNS 3 : 185.199.110.153
    • DNS 4 : 185.199.111.153
  4. Période : 12 mois (gratuit)
  5. Clique "Continue"
  6. Finalise la commande
4.4
Configurer le domaine avec GitHub
  1. Retourne sur ton repository GitHub
  2. Va dans Settings → Pages
  3. Dans "Custom domain", écris ton nouveau domaine
    Exemple : monsite.tk
  4. Clique "Save"
  5. ✅ Coche "Enforce HTTPS" (attends 5 minutes avant)
4.5
Tester ton domaine
⏱️ Patience : Ça peut prendre 24-48h pour que tout fonctionne parfaitement.

Test :

  • Tape ton domaine dans le navigateur
  • Si ça ne marche pas immédiatement, c'est normal !

🆘 ÉTAPE 5 : SUPPORT ET DÉPANNAGE

Problèmes courants et solutions

❌ Mon site ne s'affiche pas sur GitHub

Solutions :

  1. Vérifie que ton fichier s'appelle exactement index.html
  2. Attends 10-15 minutes
  3. Vide le cache de ton navigateur (Ctrl+F5)
  4. Vérifie que GitHub Pages est bien activé dans Settings

❌ Mon domaine Freenom ne fonctionne pas

Solutions :

  1. Attends 24-48h (c'est normal)
  2. Vérifie que tu as bien mis les DNS GitHub
  3. Assure-toi que le domaine est bien configuré dans GitHub Settings

❌ Mon site ne s'affiche pas bien sur mobile

Solutions :

  1. Demande à Claude de corriger le responsive
  2. Ajoute cette ligne dans ton HTML (entre les balises <head>) :
<meta name="viewport" content="width=device-width, initial-scale=1.0">

❌ Je veux changer quelque chose sur mon site

Solution :

  1. Va sur Claude AI
  2. Demande la modification
  3. Copie le nouveau code
  4. Met à jour ton fichier index.html sur GitHub
